## 1. Run Locally (No Real Money, No External Services)

This is the fastest way to see the full system working. Everything runs on your machine.

### What happens

- Hardhat spins up a local blockchain (chain ID 56)
- `deploy-local.ts` deploys **MockUSDT + MockAave + all 5 protocol contracts**
- 5 test wallets get **100,000 fake USDT** each
- Addresses are **auto-written** to `lib/contracts/addresses.ts` ‚Äî no copy-pasting
- Frontend reads from localhost via `NEXT_PUBLIC_USE_LOCAL=true`

### Commands (3 terminals)

**Terminal 1 ‚Äî Blockchain node:**

```bash
cd contracts
npx hardhat node
```

Leave running. You'll see `Started HTTP and WebSocket JSON-RPC server at http://127.0.0.1:8545/` and a list of 20 funded accounts.

**Terminal 2 ‚Äî Deploy everything:**

```bash
cd contracts
npx hardhat run scripts/deploy-local.ts --network localhost
```

This prints all deployed addresses and writes them to the frontend automatically.

**Terminal 3 ‚Äî Frontend:**

```bash
npm run dev
```

Open `http://localhost:3000`.

### Test the full circle lifecycle

2. Switch to Account #1 ‚Üí **Join Circle**

5. Anyone ‚Üí **Trigger Payout** (recipient gets pot minus 2% fee)
6. Account #1 ‚Üí **Exit Early** (gets 98% refund)

### Restarting

Every time you restart the Hardhat node, all state is wiped. Run the deploy script again:

```bash
# Terminal 1
cd contracts && npx hardhat node

# Terminal 2
cd contracts && npx hardhat run scripts/deploy-local.ts --network localhost

# Terminal 3
npm run dev
```

Also clear MetaMask activity: Settings ‚Üí Advanced ‚Üí Clear Activity Tab Data (prevents nonce errors).

## 3. Forked BSC Testing (Real Aave V3, Real USDT)

For testing with **real Aave V3 yield**, you fork BSC mainnet locally. You still don't spend real money ‚Äî the local node mirrors real BSC state.

### Why you need an archival RPC

Free public BSC RPCs don't support forking:

| Free RPC | Error |
|----------|-------|
| `bsc-dataseed1.binance.org` | `missing trie node` (non-archival) |
| `rpc.ankr.com/bsc` | Requires API key |
| `bsc-rpc.publicnode.com` | `state histories haven't been fully indexed yet` |

### Recommended: NodeReal (BSC Official Partner)

| | Details |
|---|---|
| **URL** | [nodereal.io](https://nodereal.io/) |
| **Free tier** | 100M compute units/month |
| **Paid** | From $29/month |
| **Why** | Official BNB Chain partner, best BSC compatibility |

**Setup:**

1. Go to [nodereal.io](https://nodereal.io/) ‚Üí Sign up (free)
2. Dashboard ‚Üí API Keys ‚Üí Create Key ‚Üí Select **BSC Mainnet**
3. Copy the HTTPS endpoint: `https://bsc-mainnet.nodereal.io/v1/YOUR_KEY`
4. Create `contracts/.env`:

```env
BSC_RPC_URL=https://bsc-mainnet.nodereal.io/v1/YOUR_KEY
FORK_BSC=true
```

5. Start forked node:

```bash
cd contracts
npx hardhat node
```

Now the local node has real USDT at `0x55d398...`, real Aave Pool, and you can impersonate any USDT whale.

### Other archival RPC providers

| Provider | Free Tier | Paid | URL |
|----------|-----------|------|-----|
| **NodeReal** | 100M CU/mo | $29/mo | [nodereal.io](https://nodereal.io/) |
| **Chainstack** | 3M req/mo | $29/mo | [chainstack.com](https://chainstack.com/) |

### Performance tip

Pin a block number in `hardhat.config.ts` to enable local caching (much faster after first run):

```typescript
forking: {
  url: process.env.BSC_RPC_URL,
  blockNumber: 45000000,
  enabled: !!process.env.FORK_BSC,
},
```

## 4. Deploy to BSC Testnet

**Prerequisites:**
- Deployer wallet with tBNB (get from [testnet.bnbchain.org/faucet-smart](https://testnet.bnbchain.org/faucet-smart))
- Private key in `contracts/.env`

```env
DEPLOYER_PRIVATE_KEY=0xYOUR_KEY
```


> **Note:** BSC Testnet has no Aave V3. Aave calls will revert. Test non-yield features only (join, contribute, payout, split).

After deploying, manually update `lib/contracts/addresses.ts` with the printed addresses under chain ID `97`.

## 5. Deploy to BSC Mainnet

**Pre-flight checklist:**

- [ ] Deployer wallet funded with ~0.5 BNB for gas
- [ ] Contract code audited (see `MOTSHELO_V2_AUDIT.md`)
- [ ] All unit tests passing (`npx hardhat test`)
- [ ] Full lifecycle tested on local dev
- [ ] Chainlink VRF subscription funded with LINK (if using RANDOM payout order)
- [ ] BscScan API key ready for contract verification

```env
DEPLOYER_PRIVATE_KEY=0xYOUR_PRODUCTION_KEY
BSC_RPC_URL=https://bsc-mainnet.nodereal.io/v1/YOUR_KEY
BSCSCAN_API_KEY=YOUR_BSCSCAN_KEY
```

```bash
cd contracts
npx hardhat run scripts/deploy.ts --network bsc
```

### Verify contracts on BscScan

```bash
npx hardhat verify --network bsc 0xFEE_COLLECTOR_ADDRESS "0xDEPLOYER_ADDRESS"
```

For Factory (many constructor args), create `scripts/factory-args.ts` with the constructor values and use:

```bash
npx hardhat verify --network bsc 0xFACTORY_ADDRESS --constructor-args scripts/factory-args.ts
```

### Post-deploy wiring

The deploy script handles steps 1-7 automatically. Step 8 is manual:

1. Go to [vrf.chain.link](https://vrf.chain.link/) ‚Üí BSC Mainnet
2. Fund your VRF subscription with LINK tokens
3. Add the Factory contract address as a consumer

Update `lib/contracts/addresses.ts` with all deployed addresses under chain ID `56`.

## Troubleshooting

| Problem | Fix |
|---------|-----|
| **$0.00 balance on local dev** | You're using your personal wallet. Import a Hardhat test account (see Section 1). |
| **"Can't create circle"** | Make sure contribution ‚â• 10 USDT. On local dev, use test accounts with 100k USDT. |
| **`npm install` auth error** | `npm logout` ‚Üí `npm config set registry https://registry.npmjs.org/` ‚Üí retry |
| **`nonce too high`** | Restart Hardhat node + MetaMask ‚Üí Settings ‚Üí Advanced ‚Üí Clear Activity Tab Data |
| **MetaMask stuck transaction** | Same as above ‚Äî clear activity tab data |
| **`missing trie node` when forking** | Free BSC RPCs don't support forking. Use NodeReal (free tier). |
| **`FORK_BSC` not working (Windows)** | PowerShell: `$env:FORK_BSC="true"` / CMD: `set FORK_BSC=true` |
| **Frontend shows wrong addresses** | Re-run `deploy-local.ts` ‚Äî it auto-writes `addresses.ts` |
| **`dependency @metamask/sdk not found`** | `npm install @metamask/sdk --legacy-peer-deps` in root dir |
| **Aave reverts on BSC testnet** | Testnet has no Aave V3. Use local dev (mocks) or forked mainnet. |
| **Payout math looks wrong** | USDT on BSC uses 18 decimals (not 6 like USDC). Use `parseEther`/`formatEther`. |
